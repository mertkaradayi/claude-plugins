# X Algorithm Tweet Optimizer

Generate algorithm-optimized tweet versions from your idea.

## Usage

- `/tweet <your idea>` - Generate optimized tweet versions
- `/tweet thread: <topic>` - Generate an optimized thread
- `/tweet analyze: <draft>` - Analyze an existing draft
- `/tweet hooks` - Show available hook templates

## What You Get

- 3-4 tweet versions using different hooks
- Algorithm grade (A-F) for each version
- SimClusters alignment score
- Engagement probability
- Timing & media recommendations
- Post-tweet action checklist
